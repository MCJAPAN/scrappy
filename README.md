# SCRAPPY (スクラッピー)

シンプルかつ高速なマルチファイルダウンローダーです。
連番URL・URLリスト・ドラッグ＆ドロップに対応しています。

---

## ✨ 特徴

* 🔢 連番URLダウンロード（`{:04d}` などに対応しています。）
* 📄 URLリスト読み込み（.txt / .csv に対応しています。）
* 🖱️ ドラッグ＆ドロップで簡単に操作できます。
* ⚡ 並列ダウンロードで高速に処理できます。
* 🔁 自動リトライ機能で失敗にも対応します。
* 📊 進捗バーで状況が分かりやすいです。
* 🧠 拡張子を自動判定します（jpg / png / mp4 / mp3 / html / txt など。）
* 💾 設定をJSONで保存できます。

---

## 📦 ダウンロード

Releases から最新版を取得してください。
（`SCRAPPY.exe`）

---

## 🚀 使い方

### ① 連番URLの場合

例：

```id="h2kq6s"
https://example.com/data/{:04d}.jpg
```

| フォーマット   | 説明                |
| -------- | ----------------- |
| `{:04d}` | 0001 のように4桁になります。 |
| `{:03d}` | 001 のように3桁になります。  |
| `{:02d}` | 01 のように2桁になります。   |
| `{:d}`   | 1 のようにそのままになります。  |

---

### ② URLリストの場合

1. `.txt` にURLを1行ずつ記述します。
2. ファイルをドラッグ＆ドロップします。
3. ダウンロードを開始します。

## ⚙️ 設定項目

* ベースURLです。
* 開始番号 / 終了番号です。
* 保存フォルダです。
* 同時ダウンロード数です。
